CANON LBP-810 ДЛЯ WINDOWS 11 x64
================================

Версия пакета: 1.0.0

Требования:
- Windows 11 x64;
- Canon LBP-810, подключённый напрямую по USB;
- бумага A4;
- права администратора во время установки.

Установка:
1. Полностью распакуйте ZIP в любую папку.
2. Подключите и включите Canon LBP-810.
3. Дважды щёлкните Install.cmd и подтвердите запрос Windows.
4. После завершения запустите Print-TestPage.cmd.

После установки архив и распакованную папку можно удалить. Рабочие файлы
копируются в:
  C:\Program Files\Canon LBP-810 Bridge

В Windows появится обычная очередь:
  Canon LBP-810

Виртуальная машина, режим совместимости и оригинальный 32-битный драйвер
Canon не требуются.

Удаление:
  Запустите Uninstall.cmd из распакованного пакета.

Диагностика:
  C:\ProgramData\Canon LBP-810 Bridge\spool\bridge.log

Ограничения версии 1.0.0:
- только Windows 11 x64;
- только USB VID_04A9 / PID_260A (Canon LBP-810);
- монохромная печать, 600 dpi;
- основной проверенный формат бумаги — A4.

Исходный код и обновления:
